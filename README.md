## Quick Start

1. Browse the documentation in the language of your choice
2. Start with **Getting Started** for an introduction
3. Refer to **API Reference** for detailed function documentation
4. Check **Plugin Development** for creating custom Python plugins

## Requirements

- CRYENGINE 5.7 or later
- Python 3.7 (the version embedded by CRYENGINE 5.7)
